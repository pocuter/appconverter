# APPConverter

A converter to convert a ESP32 flash file into a pocuter app

appconverter.exe

    -image   [image filename]
   
    -version [X.Y.Z]
   
    -meta    [metadata filename]
   
    -sign    [private key filename]
   
    -id      [APP-ID]
   
    -help


Meta file example (it's an ini file):

[APPDATA]<br/>
Name=DOOM II<br/>
Author=id Software<br/>
